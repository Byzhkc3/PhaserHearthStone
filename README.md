# PhaserDemo

---

学习Phaser游戏开发的小demo

### 1.使用方法

确保你的机器已经安装了node环境。

将源码下载至本地，使用

```
npm install
```

安装相关依赖。然后用

```
gulp default
```

启动游戏服务。
